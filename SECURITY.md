# Security

## Reporting a vulnerability

Report security issues **privately** so details are not disclosed before a fix is available:

- Use **[GitHub Security Advisories](https://github.com/hemanthkumar650/ERP-GenAI-Knowledge-Assistant/security/advisories/new)** for this repository (preferred), or  
- Contact the repository owner through a **private** channel.

Please do **not** open a public issue for undisclosed vulnerabilities.

## Secrets and configuration

- Do **not** commit `.env` or real API keys. Use `.env.example` as a template only.
- Rotate any credential that may have been exposed (e.g. committed by mistake or shared in logs).
